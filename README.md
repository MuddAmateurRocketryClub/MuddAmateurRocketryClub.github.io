# Mudd Amateur Rocketry Club

SpaceX-inspired redesign of the MARC club website

## Design reference

[SpaceX Website — Home Page (Community) on Figma](https://www.figma.com/design/sctOCsZKsFHA48CsnkN7X0/SpaceX-Website---Home-Page--Community-?node-id=1-81)

## Run locally

Requires Node.js 20.19.0 or newer.

```bash
npm ci
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Pages

- `/` — Home
- `/about` — About, teams, leadership
- `/rockets` — Flight heritage
- `/join` — FAQs
- `/contact` — Contact form
- `/donate` — Sponsorship
